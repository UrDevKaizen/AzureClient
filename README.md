## Building

### Requirements

- Java 21+
- Gradle
- IntelliJ IDEA (Recommended)

### Steps

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Allow Gradle to download dependencies.
4. Run the `build` Gradle task.
